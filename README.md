## 🧑‍💻 Proyectos Destacados

### 🎯 Sistema Integral de Gestión Educativa (Full-Stack)
Un ecosistema completo para la gestión de alumnos, materias y calificaciones, diseñado con una arquitectura desacoplada y preparada para producción.

* **[Backend (API REST)](https://github.com/JaimeJaker/student-grades-api-backend.git):** Construido con **Java 17** y **Spring Boot 3.5**. Implementa persistencia con **Spring Data JPA**, validación de datos y base de datos **MySQL**.
* **[Frontend (SPA)](https://github.com/JaimeJaker/student-grades-api-frontend.git):** Interfaz moderna desarrollada con **React 18** y **TypeScript**. Utiliza **Vite** para un desarrollo ágil, **Axios** para el consumo de la API y **React Router DOM** para la navegación.
* **Infraestructura:** Despliegue orquestado con **Docker Compose**, utilizando **Nginx** como servidor de estáticos.

### 🔐 FortiFile
Aplicación de escritorio enfocada en la seguridad de la información.
* **Propósito:** Gestión segura de archivos confidenciales mediante criptografía.
* **Stack:** Python, PyQt5, SQLite, Cryptography, bcrypt.
* **Contribución:** Desarrollo del motor de cifrado, lógica de autenticación y testing con `pytest`.
* **Repositorio:** [FortiFile en GitHub](https://github.com/andrefalar/Software-Engeneering-1)

### ⚙️ Proyectos de Sistemas y Embebidos
* **Administrador de Archivos Cifrados:** Herramienta de sistema para protección de datos local.
* **Proyectos ESP32:** Desarrollo de lógica embebida con **FreeRTOS**, control de periféricos y pantallas OLED.
