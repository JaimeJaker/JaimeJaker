## 🎓 Sistema de Gestión Educativa (Full Stack)

Proyecto completo que implementa una arquitectura cliente-servidor con separación clara entre frontend y backend.

---

### 🔧 Backend - API REST

🔗 https://github.com/JaimeJaker/student-grades-api-backend.git  

API RESTful desarrollada con **Spring Boot**, diseñada para gestionar:

- 👨‍🎓 Estudiantes  
- 📚 Materias  
- 📝 Notas  

**Características:**

- CRUD completo para cada entidad  
- Validación de datos con Spring Validation  
- Persistencia con JPA + Hibernate  
- Arquitectura en capas (controller, service, repository)  
- Contenerización con Docker  

**Stack tecnológico:**

- Java 17  
- Spring Boot  
- Spring Data JPA  
- Hibernate  
- MySQL  
- Maven  
- Docker / Docker Compose  

---

### 🎨 Frontend - SPA

🔗 https://github.com/JaimeJaker/student-grades-api-frontend.git  

Aplicación web tipo **Single Page Application (SPA)** que consume la API backend.

**Características:**

- Interfaz modular y tipada  
- Consumo de API mediante Axios  
- Navegación con React Router  
- Manejo de estado y formularios  
- Diseño desacoplado del backend  

**Stack tecnológico:**

- React 18  
- TypeScript  
- Vite  
- Axios  
- React Router DOM  
- Nginx (despliegue en Docker)  

---

## 🧩 Arquitectura del sistema

`React + TypeScript (Frontend)` → `HTTP (Axios)` → `Spring Boot API (Backend)` → `JPA / Hibernate` → `MySQL`
